# AnA_NatsuYasumi
### 만들어진 것 - 윾니티 리듬게임
AnA 여름방학 개인 프로젝트  

## 도전해본 아이디어
* 간단한 가상머신 - 될지는 모르겠음 아마 안될듯, 망함
* 존나 허접한 운영체제 - 최후 직전의 보루, 살아있음
* DxLib - 난생 처음 배워보는 게임 프로그래밍, 최후의 보루 그자체
* Unity로 리듬게임 - 될때까지 비벼보는 중
